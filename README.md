# 儿童图书推荐系统 python1165

## 项目概述

本项目是一个基于协同过滤算法的儿童图书推荐系统，采用前后端分离的技术架构。适用于项目实训、在线课程任务等多种场景。

## 技术栈

- **前端**：Vue、HTML、CSS、JavaScript、ElementUI
- **后端**：Python、Django、MySQL
- **开发环境**：VSCode、PyCharm、Navicat

## 功能模块

### 系统角色

系统主要分为以下两个角色：

- **管理员**
  - 用户管理
  - 图书分类管理
  - 儿童图书管理
  - 热销图书管理
  - 公告信息管理
  - 系统管理
  - 订单管理
  - 我的个人信息管理

- **用户**
  - 个人中心
  - 修改密码
  - 我的订单
  - 我的地址
  - 我的收藏

## 运行环境

- 建议配置：
  - Python 3.8+
  - Django 3.x
  - Vue 2.x
  - MySQL 5.7+

## 部署步骤

1. 安装Python及相关依赖
2. 安装并配置MySQL数据库
3. 部署后端服务
4. 部署前端服务
5. 启动系统
